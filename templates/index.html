Exploratory Data Analysis & Analytics

Model performance graphs, feature correlations, and dataset summaries

Feature Correlation Map

This Pearson correlation matrix tracks linear relationships between all numerical features in the dataset.

  • Vehicle Density & Speed: Strong negative correlation. As density increases, average vehicle speed decreases, pointing to congestion.
  • Flow Rate: Heavily correlated with vehicle density, since flow is density multiplied by speed.
  • Network Metrics: Show correlation between latency and packet loss, representing edge processing network quality.

Traffic Condition Distribution

Demonstrates target variable balance across 4,000 recorded samples:

  • Medium: The most prevalent state in typical city streets.
  • Low & High: Represents off-peak open highways and peak congestion periods, respectively.
  • The balanced distribution ensures classifier accuracy is not biased towards a single majority class.

Speed vs. Vehicle Density

Visualizing the separation boundaries between Low, Medium, and High classes:

  • Low traffic forms in the top-left region (high average speed, low density).
  • High congestion is clustered in the bottom-right region (high density, low speed).
  • Medium traffic populates the transition boundary. A non-linear machine learning algorithm is required to separate these overlapping domains accurately.

Neural Network Learning Curve

PyTorch MLP performance over epochs:

  • Loss curves: Training and Validation losses decay smoothly. Overfitting is minimized using standard regularizations: Batch Normalization, Dropout (0.2), and Weight Decay (L2 regularization).
  • Accuracy curves: Illustrates validation accuracy converging steadily.
  • Learning rate scheduler automatically decays learning rate upon plateauing, optimizing validation performance.
